Ревью сессии

Каждую неделю на нашем телеграм канале проводятся ревью сессии, в которых может участвовать любой желающий.

Все наши сессии проходят в рамках вашего Github репозитрия, так что вам не придется никуда загружать или выкладывать свой код.

Ревью можно получить для любого языка или фреймворка.
